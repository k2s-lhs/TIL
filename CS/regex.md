# 자주 사용하는 정규식
[regexr :: 정규식 검증 사이트](https://regexr.com/)

1. 영문자 대소문자, 숫자 (공백 X)
    
    ```java
    
    ^[a-zA-Z0-9]*$
    ```
    
2. 영문자 대소문자, 숫자 (공백 O)
    
    ```java
    ^[a-zA-Z0-9 ]*$
    ```
    
3. 한글, 대문자, 특수문자, 공백
    
    ```java
    ^(?=.*[ㄱ-ㅎㅏ-ㅣ가-힣A-Z!@#$%^&*()_+\-=\[\]{};':"\\|,.<>\/?])
    ```
    
4. Y, N 만 허용하는 정규식
    
    ```java
    ^[YN]$
    ```
    
5. 연월일 체크 
    
    ```java
    ^(19|20)\d\d([- /.])(0[1-9]|1[012])\2(0[1-9]|[12][0-9]|3[01])$
    ```
    
6. 일반 전화번호
    
    ```java
    ^\d{2,3}-\d{3,4}-\d{4}$
    ```
    
7. 휴대폰 번호
    
    ```java
    // 휴대폰 번호
    ^\d{3}-\d{3,4}-\d{4}$/
    ```
